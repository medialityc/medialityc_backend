Entidad: Area
- Endpoint: GetAllAreaEndpoint
	Ruta: GET /areas/all
	Descripcion: Listado paginado de areas con filtro opcional por nombre.
	Request (query GetAllAreaRequest):
		- Page (int?, minimo 1)
		- PageSize (int?, minimo 1, maximo 200)
		- Search (string?, texto a buscar en nombre)
		- SortBy (string?, propiedad para ordenar, por defecto Id)
		- IsDescending (bool?, orden descendente si es true)
	Response (PaginatedResponse<GetAllAreaResponse>):
		- Data[]: Id (int), Name (string)
		- Page (int)
		- PageSize (int)
		- TotalCount (int)
		- TotalPages (int)
	Autenticacion: Si

- Endpoint: GetAreaByIdEndpoint
	Ruta: GET /areas/{Id}
	Descripcion: Obtiene los datos de un area especifica por su identificador.
	Request (ruta GetAreaByIdRequest):
		- Id (int, requerido)
	Response (GetAreaByIdResponse):
		- Id (int)
		- Name (string)
	Autenticacion: Si

- Endpoint: CreateAreaEndpoint
	Ruta: POST /areas/create
	Descripcion: Crea un nuevo registro de area verificando duplicados por nombre.
	Request (JSON CreateAreaRequest):
		- Name (string, requerido, maximo 200 caracteres)
	Response (GenericAreaResponse):
		- Id (int)
		- Name (string)
	Autenticacion: Si

- Endpoint: UpdateAreaEndpoint
	Ruta: PUT /areas/update
	Descripcion: Actualiza el nombre de un area existente validando duplicados.
	Request (JSON UpdateAreaRequest):
		- Id (int, requerido)
		- Name (string?, maximo 200 caracteres)
	Response (GenericAreaResponse):
		- Id (int)
		- Name (string)
	Autenticacion: Si

- Endpoint: DeleteAreaEndpoint
	Ruta: DELETE /areas/delete
	Descripcion: Elimina un area existente por identificador.
	Request (JSON DeleteAreaRequest):
		- Id (int, requerido)
	Response: mensaje de confirmacion (string)
	Autenticacion: Si

- Endpoint: GetAreaByIdLandingEndpoint
